Stuff I've Made

  • ClenchRex

    Python program built using OpenCV and PyAutoGUI libraries to play games like Google Dinosaur and Flappy Birds using hand-gestures. Developed to enhance hand-related physiological rehabilitation exercises.

  • AirSnare

    AirSnare is a Computer Vision program built using Python. AirSnare lets you play Air Drums using two coloured drum sticks. Grab a pair of drum sticks (or sharpies), and get ready to play your favourite drum solo!

  • VisuaLooper

    VisuaLooper is a Python program that detects the tilt of your guitar's headstock and loops guitar audio based on the guitar's tilt. Audio controls are executed using Audacity - a free audio editing software.

  • Bob's Little Free Library

    Bob's Little Free Library is a book sharing Android app created as the term project for CMPUT 301 - Intro to Software Engineering in Fall of 2020. The app is for a community of readers that want to upload, borrow, lend, and exchange each other's books with ease.

  • Visualizing Static Analysis for Android Applications

    A research project for CMPUT 416/500 - Foundations of Program Analysis which combines Educational Technology and Static Analysis. This project includes a Web-based visualization for the qualitative static analysis tool - COVA.

  • Hand Gesture Recognition

    Python program that tracks hand gestures. Currently using this script to build a personalized smart mirror.
